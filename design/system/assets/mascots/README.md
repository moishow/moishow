# 마스코트 (래스터 PNG)

레퍼런스 3D 젤리 이미지에서 **각 캐릭터를 개별 추출**한 투명 PNG(알파 채널)다. **제품 마스코트의 정식(production) 에셋**이다.

| 파일                | 캐릭터              | 비고        |
| ------------------- | ------------------- | ----------- |
| `mascots.png`       | 마스코트 단체       | 활기        |
| `mascot_pink.png`   | 핑크 (꼬리 끈 포함) | 미소        |
| `mascot_red.png`    | 코랄                | 윙크        |
| `mascot_blue.png`   | 블루 (주연)         |   미소          |
| `mascot_green.png`  | 그린                | 차분한 미소 |
| `mascot_yellow.png` | 옐로                | 활짝 웃음   |
| —                   |

## 용도

- 이 PNG를 제품 마스코트로 **직접 사용**한다(이모지 금지 원칙). 온보딩·빈 상태·정산 완료 축하 화면에만 등장.
- 등장 허용/금지 규칙은 [`../../DESIGN.md`](../../DESIGN.md) 6절 마스코트 가이드를 따른다.
