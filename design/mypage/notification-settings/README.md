# 알림 설정

> 화면: [notification-settings.html](notification-settings.html) · 시나리오: [알림 설정](../../../docs/scenario/user/notification-preferences/README.md)

## 역할

정산·송금 요청·미납·소셜 알림의 수신 범위를 사용자가 조정하는 화면이다.

## 핵심 기능

- 카테고리별 푸시 알림 토글
- 변경 사항 즉시 저장

## 연결

[설정](../settings/README.md)에서 진입하고 뒤로 가기로 설정으로 돌아간다.

## 논의 필요

- 송금 요청·입금 확인처럼 정산에 필수인 알림을 끌 수 없게 할지 결정한다.
