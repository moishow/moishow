# 상대 프로필

> 화면: [relative-profile.html](relative-profile.html) · 시나리오: [소셜 프로필](../../../docs/scenario/user/social-profile/README.md)

## 역할

팔로워·팔로잉 목록에서 선택한 사용자의 공개 프로필과 최근 게시글을 확인하고 팔로우 상태를 변경한다.

## 표시·상호작용

- 프로필 사진, 이름, 소개, 게시글·팔로워·팔로잉 수
- 함께한 모임 정보와 최근 게시글
- 팔로우·언팔로우 전환, 게시글 상세 진입, 목록으로 돌아가기
