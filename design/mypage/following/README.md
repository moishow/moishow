# 팔로잉

> 화면: [following.html](following.html) · 시나리오: [소셜 프로필](../../../docs/scenario/user/social-profile/README.md)

## 역할

내가 팔로우한 사용자 목록을 검색하고 언팔로우를 관리하는 화면이다.

## 핵심 기능

- 팔로잉 목록과 검색 결과·빈 상태
- 사용자 행 선택 시 상대 프로필 진입, 사용자별 팔로우 상태 표시 및 언팔로우
- 팔로워 탭 전환

## 연결

마이페이지의 팔로잉 수에서 진입하며, 팔로워 화면과 상호 이동한다.

## 논의 필요

- 언팔로우 확인 단계와 DM 가능 조건의 연계 정책을 정한다.
