# 계좌 등록·인증

> 화면: [bank-account.html](bank-account.html) · 시나리오: [계좌 등록·인증](../../../docs/scenario/user/bank-account-verification/README.md)

## 역할

정산을 받을 계좌를 등록하고 예금주·인증 상태를 확인하는 화면이다.

## 핵심 기능

- 등록 계좌와 인증 상태 표시
- 은행·계좌번호 입력 및 인증 코드 확인
- 계좌 변경 또는 재인증

## 연결

[계정 설정](../account-settings/README.md)에서 진입한다.

## 논의 필요

- 계좌 인증 방식과 외부 본인 인증 서비스 연동 책임을 인증 도메인과 조율한다.
