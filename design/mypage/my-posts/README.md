# 내 게시글

> 화면: [my-posts.html](my-posts.html) · 시나리오: [소셜 프로필](../../../docs/scenario/user/social-profile/README.md)

## 역할

내가 작성한 쇼 게시글만 모아 보여 주는 소셜 조회 화면이다.

## 핵심 기능

- 내 게시글의 미디어 미리보기와 반응 수 표시
- 게시글 선택 시 기존 쇼 상세(W-33)로 이동

## 연결

마이페이지의 게시글 통계에서 진입하고, 각 카드에서 쇼 상세로 이동한다.

## 논의 필요

- 게시글 삭제·공개 범위 변경·정렬은 쇼 도메인의 책임으로 둘지 정한다.
