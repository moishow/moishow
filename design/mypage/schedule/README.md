# 내 모임 일정

> 화면: [schedule.html](schedule.html) · 시나리오: [내 모임 일정](../../../docs/scenario/user/schedule/README.md)

## 역할

내가 신청하거나 참여하는 모임 일정을 월간 달력과 다가오는 일정 목록으로 확인하는 화면이다.

## 핵심 기능

- 이전·다음 달 이동과 일정 마커
- 다가오는 일정의 날짜·시간·장소 표시
- 날짜 선택 시 해당 날짜 목록 필터링 및 일정 없는 날짜의 빈 상태
- 일정 선택 시 기존 모임·차수 상세(W-11) 진입

## 연결

마이페이지의 일정 전체에서 진입하고, 일정 행은 모임 상세로 이동한다.

## 논의 필요

- 승인 대기 일정과 확정 일정을 구분하는 표기, 일정 중복 표시 방식을 정한다.
