# 계정 설정

> 화면: [account-settings.html](account-settings.html) · 시나리오: [계정 관리](../../../docs/scenario/user/account-management/README.md)

## 역할

연결된 로그인·전화번호·수취 계좌를 확인하고 로그아웃 또는 회원 탈퇴를 시작하는 계정 관리 화면이다.

## 핵심 기능

- 카카오 로그인과 전화번호 인증 상태 표시
- 계좌 등록·인증 화면 진입
- 로그아웃 및 회원 탈퇴 전 확인 단계

## 연결

[설정](../settings/README.md)에서 진입하며 계좌 등록·인증으로 이동한다. 로그아웃은 온보딩(W-01)으로 돌아간다.

## 논의 필요

- 미송금·받을 돈·진행 중 일정이 남은 상태의 탈퇴 허용 및 데이터 보관 정책을 정한다.
