# 프로필 편집

> 화면: [profile-edit.html](profile-edit.html) · 시나리오: [프로필 조회·편집](../../../docs/scenario/user/profile-edit/README.md)

## 역할

사용자가 공개 프로필 정보를 수정하고 마이페이지에 즉시 반영하는 화면이다.

## 핵심 기능

- 프로필 사진, 닉네임, 자기소개 편집
- 성별·연령대·연락처 입력 및 저장
- KYC 실명·신용등급처럼 편집 불가 정보는 수정 대상에서 제외
- 자기소개 글자 수 표시, 사진 파일명 확인, 미저장 변경사항 이탈 경고

## 연결

마이페이지에서 진입하며 저장 또는 뒤로 가기 후 마이페이지로 돌아간다.

## 논의 필요

- 연락처를 공개 프로필에서 다룰지, 인증 정보로만 보관할지 결정한다.
