# 마이페이지 화면 목록

마이페이지 도메인의 화면 명세와 와이어프레임 모음이다. 각 폴더의 `README.md`는 화면 역할, 필수 기능, 연결 흐름, 상호작용을 기록한다.

| 영역 | 화면 |
|---|---|
| 허브 | [마이페이지](mypage/README.md) |
| 프로필·소셜 | [프로필 편집](profile-edit/README.md), [내 게시글](my-posts/README.md), [팔로워](followers/README.md), [팔로잉](following/README.md), [상대 프로필](relative-profile/README.md) |
| 일정 | [내 모임 일정](schedule/README.md) |
| 설정 | [설정](settings/README.md), [알림 설정](notification-settings/README.md), [계정 설정](account-settings/README.md), [계좌 등록·인증](bank-account/README.md) |

모든 화면은 [`design/system/AGENTS.md`](../system/AGENTS.md)의 디자인 시스템 규칙을 따른다.

## 페이지별 표시·상호작용

| 페이지 | 사용자에게 보여 줄 것 | 사용자의 상호작용 |
|---|---|---|
| 마이페이지 | 프로필, 정산 요약, 일정, 최근 내역 | 각 섹션·설정·프로필 편집으로 이동 |
| 프로필 편집 | 사진, 닉네임, 소개, 기본 정보 입력값 | 사진 변경, 정보 입력, 저장 |
| 내 게시글 | 내가 작성한 게시글 카드와 반응 수 | 게시글 상세 열기 |
| 팔로워 | 팔로워 목록, 검색창, 팔로우 상태 | 검색, 맞팔로우, 팔로잉 탭 전환 |
| 팔로잉 | 팔로잉 목록, 검색창, 팔로우 상태 | 검색, 언팔로우, 팔로워 탭 전환 |
| 상대 프로필 | 공개 프로필, 함께한 모임, 최근 게시글 | 팔로우·언팔로우, 게시글 상세, 목록으로 돌아가기 |
| 내 모임 일정 | 달력, 일정 마커, 다가오는 일정 | 날짜 선택 후 목록 필터링, 기간 이동, 일정 상세 열기 |
| 설정 | 알림·계정·약관·앱 정보 메뉴 | 각 설정 화면 열기 |
| 알림 설정 | 알림 카테고리와 현재 수신 상태 | 카테고리별 토글 변경 |
| 계정 설정 | 연결 계정, 인증 상태, 계좌 정보 | 계좌 관리, 로그아웃, 회원 탈퇴 진입 |
| 계좌 등록·인증 | 등록 계좌, 은행·계좌번호·인증 입력 | 계좌 입력, 인증 요청·재시도 |

정산 관련 영역은 마이페이지에서 요약만 보여 주고, 기존 모임 정산 도메인 화면으로 이동한다.
