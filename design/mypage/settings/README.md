# 설정

> 화면: [settings.html](settings.html) · 시나리오: [설정](../../../docs/scenario/user/settings/README.md)

## 역할

알림, 계정, 약관, 앱 정보를 관리하는 마이페이지의 설정 진입 화면이다.

## 핵심 기능

- 알림 설정과 계정 설정 진입
- 약관·개인정보처리방침 및 앱 버전 노출

## 연결

마이페이지 상단 설정 버튼에서 진입하며 알림 설정·계정 설정으로 이동한다.

## 논의 필요

- 약관 문서의 실제 호스팅 위치와 앱 버전 갱신 방식을 정한다.
