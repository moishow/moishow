# 팔로워

> 화면: [followers.html](followers.html) · 시나리오: [소셜 프로필](../../../docs/scenario/user/social-profile/README.md)

## 역할

나를 팔로우한 사용자 목록을 검색하고, 필요하면 맞팔로우하는 화면이다.

## 핵심 기능

- 팔로워 목록과 검색 결과·빈 상태
- 사용자별 팔로우·맞팔로우 상태 표시 및 동작
- 사용자 행 선택 시 상대 프로필 진입, 팔로잉 탭 전환

## 연결

마이페이지의 팔로워 수에서 진입하며, 팔로잉 화면과 상호 이동한다.

## 논의 필요

- 차단·신고 상태를 목록과 상대 프로필에 어떻게 반영할지 정한다.
