# home / home-navigation — 탭바·FAB 라우팅 허브

## 시나리오 개요
홈 하단 **탭바(홈·피드·FAB·DM·마이)** 와 내 모임 블록을 통해 전 도메인으로 이동하는 글로벌 네비게이션. 홈이 앱의 루트 네비게이션을 소유한다.

## 페르소나
멤버 · 총무 (공통)

## 트리거
- 하단 탭바 5요소: `홈`(active) · `피드` · FAB `+` · `DM` · `마이`
- 내 모임 카드 / `전체보기`

## User Flow
1. 홈 → 탭 선택 → 해당 도메인 루트 화면
2. FAB `+` → **만들기 액션시트**(글쓰기 / 모임 만들기 / 정모 개설 / 번개) → 선택지별 화면
3. 내 모임 카드 탭 → 모임 홈(멤버용) → 정모·정산 등

## 분기 / 엣지
- **FAB RBAC(D7·확정)**: 글쓰기·모임 만들기·번개는 전원, **정모 개설은 총무만** 노출
- **모임 카드 상태별 진입**: 정산 진행/확인대기/정산 완료 badge에 따라 랜딩 위치 달라질 수 있음(정모 상세 vs 모임 홈)
- **딥링크 복귀 (A3·확정)**: 타 탭에서 홈 재선택 시 **스크롤 위치·섹션 상태 유지**(초기화 안 함). 캘린더 선택 날짜·탭 필터도 보존. 단 데이터는 포그라운드 복귀 재조회(P4)로 갱신 — *위치는 유지, 내용은 최신*. 홈 탭 아이콘 재탭(이미 홈일 때)만 최상단 스크롤로 리셋.

## GAP
- **FAB `+` 만들기 액션시트**: 글쓰기/모임 만들기/정모 개설(총무)/번개 — 신규 컴포넌트(D7 확정)
- **번개 개설·번개 상세**: crew 신규(P1) — 무소속 번개 가능
- **멤버용 '모임 홈'**: W-07은 가입 프리뷰 전용. 이미 가입한 멤버가 보는 모임 홈(정모 목록·공지·멤버·정산 현황)이 없음 → crew 도메인 핵심 GAP
- **내 모임 전체 목록**: `전체보기` 목적지 없음 → **소속 5개+에서만 노출**(D5)

## 검증 메모 (Step 2)
- ✅ [확정 D7] FAB = 만들기 허브(글쓰기/모임/정모/번개), 정모 개설만 총무 RBAC.
- ✅ [확정 D8] 개설 진입점은 FAB로 단일화 — `찾기·만들기`는 탐색 전용(중복 제거).
- ✅ [확정 D5] `전체보기`는 소속 모임 5개 이상일 때만 노출.
- 멤버용 모임 홈은 crew 도메인에서 반드시 신규 페이지로 도출 필요(2차 검증 핵심).
