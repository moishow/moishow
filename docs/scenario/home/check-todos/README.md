# home / check-todos — 오늘 챙길 일 처리

## 시나리오 개요
홈의 **"오늘 챙길 일"** 블록에서 마감 임박한 회비 액션(송금·추가정산 송금·환급 수령확인)을 즉시 처리한다. 앱은 결제/에스크로를 쓰지 않고 **송금 딥링크 + [송금완료] 버튼 + 총무 확인** 워크플로우로 회비 흐름을 추적한다. 홈은 각 멤버가 지금 해야 할 회비 행동을 큐로 모아 보여주고 crew 도메인 처리 화면으로 핸드오프한다.

## 페르소나
멤버 (송금 주체) · 총무(자기 몫 송금 시)

## 트리거
- 홈 "오늘 챙길 일" 카드의 액션 버튼
  - `송금` (예: "봄 시즌 킥오프 회비 · 1인 52,000원 · 마감 D-1")
  - `추가정산 송금` (적자 정모)
  - `환급 수령확인` (총무가 역송금한 잉여 환급)

## User Flow
### 2-A. 회비 송금 (예치 대체)
1. 홈 → "봄 시즌 킥오프 회비" 카드 `송금` 탭
2. → **W-11 정모 상세**(달성 게이지·수취 계좌 확인)
3. → **[회비 송금 화면(GAP)]** → **송금 딥링크**로 뱅킹앱 이동 → 실제 계좌이체
4. 앱 복귀 → **[송금완료]** 버튼 → 상태 **확인대기**
5. 총무가 입금 대조 → **완료 처리**(납부 확정, 게이지 반영) / **미송금 처리**(반려 → 재송금 요청)

### 2-B. 추가정산 송금 (적자 정모)
1. 홈 → "추가정산 송금" 카드 탭 → **W-22 추가정산 납부**
2. 회비 송금과 동일 플로우(딥링크 → [송금완료] → 확인대기 → 총무 확인)
3. 완료 → **W-23 정산 완료(DONE)** / 미송금 → 프로필 표시·지속 푸시(S8)

### 2-C. 환급 수령확인 (잉여 환급)
1. 홈 → "환급 수령확인" 카드 탭
2. 총무가 역방향 송금 딥링크로 보낸 잉여 환급을 실제 수령 확인
3. **[수령확인]** 버튼 → 환급 완료 기록 → **W-23 정산 완료** 반영

|---|---|---|---|
| 회비 송금 | W-11 → 회비 송금(GAP) | [송금완료]→확인대기→총무 확인 | crew |

## 정렬 규칙 (P2·확정)
오늘 챙길 일 큐 정렬: **① 마감 있는 항목(잔여시간 오름차순) → ② 마감 없는 항목(생성 최신순)**, 동점 시 금액 큰 순.
- 마감 있음: 회비 송금(D-N), 추가정산 송금
- 마감 없음: 환급 수령확인 → 하단 배치

## 상태 갱신 (P4·확정 — 폴링 미사용)
송금 확인이 **총무 수동**이라 비동기 → 홈 todo는 다음으로 갱신:
- **푸시 수신**(총무 완료/미송금 처리) 시 배지·목록 갱신
- **앱 포그라운드 복귀 시 재조회**
- **당겨서 새로고침**
- 상시 폴링은 하지 않음(총무 확인은 분~시간 단위, 실시간 불필요)

## 분기 / 엣지
- **마감 경과**: D-N 지나면 카드 상태 전환(마감/미납) → S8 엣지
- **미송금 처리(반려)**: 총무가 입금 확인 실패 → 멤버에게 "미송금, 재송금 필요" 알림 → 다시 2-A
- **확인 지연**: 송금완료 후 총무 확인이 늦으면 "확인대기" 상태 지속 → 독촉/문의(CS)
- **중복 송금**: 앱이 실제 이체를 막지 못하므로, [송금완료]는 1회 기록·상태 표시로만 관리(실입금 대조는 총무 책임)

## GAP
- **회비 송금 화면(딥링크+[송금완료])**: 구 W-12/14 대체 신규 화면
- **총무 송금 확인 화면(완료/미송금 처리)**: 신규 — manage-as-leader와 공유
- "오늘 챙길 일" 큐 소스 규칙: {회비 송금 마감 D-N, 추가정산 송금, 환급 수령확인}. 총무 전용(송금 확인·정산)은 `manage-as-leader`로 분리.

## 검증 메모 (Step 2)
- 송금 진입을 W-11(정모 상세 경유) vs 바로 송금 화면 중 무엇으로 — 금액·수취계좌 확인 필요상 **W-11 경유** 권장.
- ✅ [확정] 결제/에스크로 제거. 송금 딥링크 + [송금완료] + 총무 확인 모델.
- ✅ [확정] 추가정산·환급도 동일 송금+확인 방식.
- 미송금 처리 후 신용등급·프로필 표시 정책(S8)과의 연결 확인.
