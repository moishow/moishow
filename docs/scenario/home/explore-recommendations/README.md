# home / explore-recommendations — 추천 모임·번개 탐색·가입 진입

## 시나리오 개요
홈 하단 추천 영역은 **추천 모임 섹션**과 **추천 번개 섹션**으로 분리된다(P1·확정). 추천 태그·추천 모임에서 새 모임을 발견해 탐색(W-06)/가입(W-07~09)으로, 추천 번개에서 즉흥 참여 대상을 발견해 **번개 상세**로 진입한다. 친분 없는 모임/번개도 홈에서 발견 → 검증 → 참여로 잇는 콜드스타트 획득 경로.

## 페르소나
멤버 (가입·참여 희망자)

## 트리거
- 추천 태그 chip: `내 주변 모임` · `당장 가입 가능` · `이번 주 활동` · `관심사 맞춤`
- **추천 모임 섹션** 카드의 `가입` 버튼 (예: "새벽 러닝크루")
- **추천 번개 섹션** 카드 (예: "오늘 저녁 번개 풋살") — 탭 시 번개 상세
- 내 모임 블록의 `찾기·만들기`(+ 점선 카드) — 탐색 측 진입

## User Flow
### 5-A. 태그로 탐색
1. 홈 → 추천 태그 chip 탭 → **W-06 모임 탐색**(해당 필터 적용)
2. 목록에서 모임 선택 → **W-07 모임 상세(안심예산안)**
3. → **W-08 가입 신청서(폼빌더)** → **W-09 신청 완료/대기**
4. 총무 승인 → **W-26 알림** → 멤버십 생성 → 홈 "내 모임"에 반영

### 5-B. 추천 모임 바로 가입
1. 홈 → 추천 모임 `가입` 탭 → **W-07 모임 상세**(프리뷰 확인 강제)
2. 이후 W-08 → W-09 → W-26 (5-A와 합류)

### 5-C. 추천 번개 참여 (P1·확정)
1. 홈 → 추천 번개 카드 탭 → **[번개 상세(GAP)]** (정모 상세 W-11과 유사한 화면)
2. 참여 신청 → (회비 있으면) 회비 송금(check-todos 2-A와 동일)
3. **무소속 번개**(모임에 속하지 않은 번개)는 모임 홈이 없으므로 번개 상세가 종점 컨텍스트

## 분기 / 엣지
- **콜드스타트(D6·확정)**: 내 모임 0인 신규 사용자에게만 추천 모임/번개 카드 노출, 모임 보유 후엔 **태그 씨앗만** 유지
- **즉시 가입 가능 vs 승인 필요**: "당장 가입 가능" 태그 모임은 신청서 없이 바로 멤버십? *(논의: W-08 폼빌더 항상 경유 vs 스킵)*
- **이미 가입한 모임 추천 노출**: 중복 방지 규칙 필요
- **가입 반려**: W-09 대기 → 반려 알림(W-26)
- **무소속 번개**: 모임 홈·멤버십 개념 없음 → 번개 상세에서 참여/회비만 처리

## GAP
- **번개 상세 화면**: 정모 상세(W-11)와 유사하나 모임 종속이 아닐 수 있음 — crew 도메인 신규(P1)
- **추천 번개 섹션**: 추천 모임과 별도 섹션(홈 컴포넌트 신규)
- `가입` 버튼이 W-07(상세)을 반드시 경유하는지 — 안심예산안 사전 확인은 핵심가치라 **상세 경유 권장**

## 검증 메모 (Step 2)
- ✅ [확정 P1] 추천 모임/추천 번개 섹션 분리, 번개 탭 → 번개 상세(GAP).
- ✅ [확정 D6] 추천 카드는 콜드스타트(내 모임 0)에서만, 이후 태그 씨앗만.
- ✅ [확정 D8] 개설 진입점은 FAB로 단일화 — `찾기·만들기`는 탐색 전용.
- 추천 로직(내 주변/관심사)의 소스는 온보딩 관심 카테고리(W-02) + 위치. 데이터 의존성 확인.
