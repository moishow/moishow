모이쇼 · 홈 도메인 와이어플로우 (신 송금 모델)

홈에서 시작하는 전체 시나리오 → 화면 흐름

docs/scenario/home의 7개 시나리오를 레인(H1~H7)으로 펼친 흐름 보드다. 결제·에스크로 없이 송금 딥링크 + [송금완료] + 총무 확인 모델을 쓴다(구 예치·동의 모델은 docs/wireframe.html). 홈 노드는 실제 화면 design/screen/homepage/homepage.html·cold-start.html, 나머지는 design/system/screens/W-xx.html을 축소 삽입했다. 아직 만들지 않은 화면(GAP)은 점선 노드로 표시 — 전부 crew·user 도메인 소관으로, 홈은 진입점만 제공한다.

카드 = 실제 화면(축소) 점선 = 미구현(GAP) 화면 →화살표 = 전환(라벨=행동/조건) 다이아몬드 = 분기 홈/정산(money) 모임/탐색(community)
