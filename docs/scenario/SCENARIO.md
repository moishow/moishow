## 1. 제품 전제

- **모바일 앱** — 모든 사용자는 로그인 상태(비로그인 플로우 없음). 딥링크 수신자가 앱 미설치면 스토어 유도.
- **돈은 앱을 거치지 않는다** — 포인트·에스크로·원장 없음(KYC 부담으로 폐기). 실제 이체는 카카오페이(외부), 앱은 **기록·확인·공개**로 신뢰를 만든다.
- **관통 원칙**: 기록은 지우지 않는다(취소·탈퇴·정정도 기록+표시 전환) · 시스템 제재보다 기록 노출 · 결정에는 근거를 적는다.
- **용어 정본**: 커뮤니티 실체=**모임**(구 동아리) · 이벤트=**정모**(정기, 모임장·관리자 개설)/**번개**(즉흥, 부원 누구나 · 내부/공개 타입). 폐기 어휘: 차수·에스크로·원장·포인트·잉여 환급.
- **역할**: **모임장**(슈퍼유저 — 생성·해산·승계·정보 수정+전권) / **관리자**(정모 주최·정산·공지·가입 승인·강퇴 등 실무) / 부원 / 비부원 참여자(공개 번개). 정산 책임은 역할이 아닌 **주최 행위에 귀속**.
- **화면 배치 축**: 홈=현황·행동(오늘 챙길 일·보낼/받을 돈·일정) / 마이=정체성·기록·설정.

## 2. 모임 도메인 (확정 · 18 시나리오/41 행위 — 담당 해찬)

### 2.1 코어: 모임 생애주기

**모임_둘러보기** — 비부원의 발견·열람
- `모임_정보_열람`: 비부원이 모임 상세를 열람 — 커버·이름·카테고리·소개 전문·멤버 인원수와 모임장/관리자 역할 표시·활동 미리보기(최근 정모/번개 2~3건). 게시글·정산 아카이브는 🔒 잠금(가입 후 공개). 가입 방식 배지(즉시/승인제) 표시. **분기**: 이미 부원→모임 홈 리다이렉트 · 승인 대기 중→CTA `승인 대기` 비활성 · 해산된 모임→안내+탐색 유도 · 블랙리스트 등록자→버튼 비활성+"이 모임에 가입할 수 없어요"(사유 미표기).
- `초대_링크_진입`: 관리자가 만든 초대 링크(기본 무기한, 수동 회수·재발급) 탭 → 앱 열림 → **"OO님의 초대"** 표시와 함께 모임 상세 진입. 회수된 링크는 무효 안내.

**모임_가입** — 상태 전이: 미신청 → (즉시: 부원) / (승인제: 작성 중→대기→승인/거절/철회)
- `폼_생성`: 승인제 선택 시 관리자가 가입폼 구성 — 질문 타입 단답/장문/객관식, **최소 1질문 강제**, **시스템 기본 필수 항목 = 모임별 닉네임**. 폼 수정 시 대기 중 신청은 제출 시점 버전으로 심사(재작성 요구 없음).
- `신청`: 즉시 가입 = 버튼 → **모임별 닉네임 1회 입력**(전역 닉네임 프리필) → 즉시 부원. 승인제 = 폼 작성(필수 검증) → 제출 → 대기. **규칙**: 중복 신청 차단 · 제출 후 응답 수정 불가(고치려면 철회→재신청) · 강퇴 이력자는 즉시 가입 모임이어도 승인제 경로 강제 · 블랙리스트는 신청 자체 차단.
- `승인`: 신청 목록(신청일순) → 폼 응답 열람 → 승인(체크박스 **일괄 승인 지원**) → 알림 → 부원 편입. 처리 이력(누가 언제) 보관·관리자만 열람. 승인 순간 철회가 겹치면 **철회 우선**.
- `거절`: 사유 입력(선택) → 알림. **재신청 쿨다운 7일**, 재신청 시 이전 응답 프리필.
- `취소(철회)`: 대기 중 본인 철회 → 응답 삭제 고지 → 미신청 복귀, **즉시 재신청 가능**(거절과 달리 쿨다운 없음).

**모임_홈** — 부원 허브
- `모임_홈_열람`: 공지(NEW)·다가오는 정모/번개·멤버 요약·최근 게시글 → 각 흐름 분기. 관리자에게만 `관리` 노출(RBAC). **모임별 닉네임 수정은 모임 내 설정에서** — 정산·명단·채팅은 항상 현재 닉네임으로 표시.
- `지난_이벤트_열람`: 종료 이벤트 목록 — **무한 스크롤+주최자 필터, 영구 보관**. 취소 이벤트도 `취소됨` 배지로 포함, 정산 없던 이벤트는 `정산 없음` 명시 → 각 정산 아카이브로.

**모임_개설**
- `모임_생성`: 이름(**유일 강제**·중복 검사)·소개·카테고리·커버(기본 이미지 세트 제공) → **가입 방식 선택(즉시/승인제)** → (승인제면 폼 구성) → 완료(첫 제안 = 부원 초대 링크 공유). 개설자=모임장. **상시 채팅방 자동 생성**.
- `모임_정보_수정`: 모임장 전용. 승인제→즉시 전환 시 **대기자 전원 자동 승인**+알림.
- `모임_해산`: 모임장 전용 — 미완(미마감) 정산 있으면 차단, 2단계 확인(부원 1인이면 간소화). 해산 후 게시글·정산 아카이브 **읽기 전용 보존**(전 부원 열람 — "해산으로 증거 인멸" 방지).

**모임_채팅** — 상시방(단톡방 대체)
- 모임당 1개 자동 생성, 부원 자동 입장(탈퇴·강퇴 시 퇴장). 표시명=모임별 닉네임. 공지 고정=관리자. 해산 시 읽기 전용 아카이브. 나간 멤버 메시지는 실명+`나간 멤버` 표시.

**운영_관리**
- `공지_관리`: 모임장·관리자 작성·수정·삭제, **개수 무제한·고정 최신순**, 게시 시 부원 알림. 공지=자유 글(시스템 알림과 중복 허용).
- `초대_링크_관리`: 생성·복사·회수·재발급(무기한). 초대 경유 신청자에겐 `초대 경유` 배지. **승인제 모임의 초대 경유자 = 필수 항목(모임별 닉네임)만 작성 + 승인 생략 → 즉시 부원**.
- `역할_관리`: 모임장이 관리자 지정/해제(수 제한 없음). 해제돼도 본인이 주최한 진행 중 정산은 마감까지 책임 유지.
- `블랙리스트_관리`: **등록 경로 2개** — 강퇴 확인 시 체크 / 사후에 나간 멤버 이력에서 선택. 등록·해제 모두 관리자 가능. 효과: 가입 신청 자체 차단(초대 링크 포함). 타 모임과 비공유.

**탈퇴_강퇴_승계**
- `모임_탈퇴`: 확인 시 미납 정산 고지(명단엔 유지) → 종료. 본인 주최 진행 중 정산은 **모임장 이관**. 참여 예정 이벤트 자동 취소. 과거 기록은 실명+`탈퇴한 멤버` 표시 유지.
- `멤버_강퇴`: 모임장·관리자 — 사유+확인(+`블랙리스트 등록` 체크 옵션) → 대상자 알림. 강퇴 이력은 타 모임에 비노출. 재가입 3단계: 일반 탈퇴=자유 / 강퇴=승인제 강제 / 블랙=차단.
- `모임장_승계`: 자발 위임(대상 지정→권한 전부 이전) · 유일 모임장 탈퇴 시 지정 강제. **잠수 모임장**: 관리자가 승계 요청 → 1주 간격 3회 알림 → 전부 무응답 시 요청 관리자에게 자동 승계.

### 2.2 정모·번개 (이벤트)

**정모_번개_개설**
- `정모_개설`: 모임장·관리자. 입력 = 제목·일시·장소·정원(선택)·설명 + **신청 마감 기한** + **노쇼 페널티 비용(선택)** + 예상 비용(선택·참고용). 완료 시 **모집 카드 자동 생성**(게시판·피드 노출)+부원 알림+**이벤트 채팅방 자동 생성**. 반복 정모는 MVP 제외.
- `번개_개설`: **부원 누구나**. 위와 동일 입력 + **타입 선택: 내부(부원만)/공개(비부원 참여 가능)**. 개설 빈도 제한 없음. 이벤트 상세에 "주최 N회" 이력 표시(긍정 신호만).
- `이벤트_공유`: 딥링크 생성→카톡 등 공유. **내부 이벤트 링크를 비부원이 열면 모임 소개로 우회**(내용 숨김+가입 유도).

**정모_번개_참여**
- `이벤트_참여`: 상세(일시·장소·주최자·참여자 명단 — 부원 간 공개·정원 게이지) → 참여 → 명단+채팅방 자동 입장. 정원 마감 시 비활성(`마감` 배지, 대기열 없음). **지각 합류 허용**(종료 전까지).
- `공개_번개_참여`: 비부원도 참여 가능(멤버십 불필요) — 노출은 인원수·주최자만(부원 명단은 가입 후). 참여 후 모임 소개 노출 = **가입 유입 경로**. 정산도 동일 플로우 참여. 노쇼 비부원 재참여 제한 없음(기록은 주최자만 봄).
- `참여_취소`: **신청 마감 기한까지 자유 취소**(명단·채팅방 퇴장, 마감 해제). 기한 후 취소·불참 = 노쇼 취급 → 페널티 대상.

**정모_번개_관리** (주최자)
- `이벤트_정보_수정`: 수정→전 참여자 알림+모집 카드 자동 갱신. **일시(날짜) 변경 = 참여 리셋**+전원 재확인(채팅방은 유지 — 재확인 유도 채널). **타입 토글(내부↔공개) 허용**+알림, 공개→내부 전환 시 기참여 비부원은 유지. **정원은 현 참여자 수 미만으로 축소 불가**.
- `이벤트_취소`: 사유(선택)→전원 알림→`취소됨` 상태(목록에 기록 유지, 카드 참여 비활성, 채팅방 즉시 읽기 전용). 정산 생성 전이면 돈 이동 없음.
- `노쇼_처리`: 종료 후 참여 명단에서 노쇼 체크→실참여 확정→정산 기준. **페널티 설정 시 노쇼자는 페널티 금액으로 정산 명단에 포함**(정산 기록에 전체 노출 — 별도 낙인 UI 없음, 돈 기록으로 공개).

**정모_번개_채팅** — 이벤트방: 개설 시 자동 생성, 참여 시 자동 입장/취소 시 퇴장 → 정산 소통 채널로 **정산 마감까지 활성** → 마감 후 읽기 전용 아카이브.

### 2.3 정산 (카카오페이 · 사후 정산 단일)

> **모델**: 주최자가 먼저 지출 → 이벤트 후 실지출 기준으로 걷는다. 사전 걷기·잉여 환급·부족분 없음. 콜백이 없으므로 **참여자 셀프 신고 + 주최자 수동 확인 2단계**(상태: 미납→확인 대기→확인, +반려). 이벤트당 정산 1건.

**정산_만들기** (주최자 = 정모는 개설한 모임장·관리자, 번개는 주최 부원)
- `정산_생성`: 실지출 항목 입력+**영수증 사진 선택 첨부**(미첨부 항목 `영수증 없음` 표시) → 참여자 확정(실참여+페널티 노쇼자) → 1/N 자동+**개인별 금액 수정** → **검증식: 실참여자 분담 합 = 실지출 합 − 노쇼 페널티 수입**(불일치 시 생성 차단) → 납부 기한 → 카카오페이 요청 생성+참여자 알림.
- `정산_수정`: 항목·금액·명단·기한 정정→전원 알림, **N차 정정 이력 표시**. 이미 납부분보다 줄면 차액은 주최자가 카카오페이로 환불(앱은 체크만). 마감 후엔 수정 불가(정정 기록 append로만).
- `정산_취소`: 무효화→전원 알림→납부분 전액 환불 체크리스트. 사유 입력 시 참여자 노출.

**납부_추적** (주최자)
- `납부_확인`: `확인 대기`(셀프 신고) 항목을 카카오페이 내역과 대조→**확인 확정**(타임스탬프 기록 — 분쟁 근거). **부분 송금은 반려 단일 처리**(부분 확인 상태 없음 → 연장 신청으로 유도).
- `납부_반려`: 대조 불일치→반려→참여자 `미납` 복귀+재송금/재신고 알림. 반복 한도 없음.
- `리마인드_발송`: 기한 경과 미납자에게 **1인 1일 1회**, D-1 사전 알림은 시스템 자동.

**회비_납부** (참여자 — 부원+비부원 동일)
- `납부_실행`: 정산 상세에서 내 몫(개인별 조정 반영)·실지출 내역·영수증 검증 → `[카카오페이로 납부]`(실패 폴백: 계좌번호 복사) → 복귀 → **`[납부했어요]` 셀프 신고** → 확인 대기.
- `반려_후_재신고`: 미송금이면 재송금, 이미 보냈으면 재신고+**송금 캡처·메모 선택 첨부**.
- `기한_연장_신청`: 전액이 어려울 때 주최자에게 **개인 단위 연장 신청**(희망 기한·사유)→승인/거절. **연장 승인 = 마감 연기 약속**: 그 기한까지 강제 마감 불가. 완납 시 최종 상태는 `확인`, 기록엔 `납부 완료·연장 N회(원기한→연장기한·납부일)` — 낙인이 아닌 과정 기록.

**정산_이의**
- `이의_제기`: 참여자가 사유(금액 오류·내역 의문·오반려 등) 제출 — **납부와 병행 가능**(먼저 내고 따질 수 있음), 반복 제한 없음. 셀프 신고·확인·반려 타임스탬프가 양측 근거.
- `이의_처리`: 주최자 — 인정→정산_수정으로 정정 / 반려→사유 회신. 이의·정정 이력은 아카이브에 표시.

**정산_마감**
- `마감_확정`: 전원 `확인` 시 활성. **강제 마감 허용**(미납자는 `미납` 기록으로 잔존 — 영원히 열린 정산 방지). 단 **연장 승인자의 연장 기한 전엔 마감 불가**. 마감=기록 잠금.
- `아카이브_열람`: **전 부원 열람**(비부원은 자기 참여분만) — 실지출 항목·영수증 원본·**인별 금액 전부 공개**·연장/정정/이의 이력·읽기 전용 잠금 표시. **모임 장부 CSV**: 관리자가 기간 선택→전체 정산 내보내기(승계 인수인계·회계 보고).

### 2.4 공통
- `알림`: 모임 도메인 트리거 15종(공지·정산 요청/기한/확인/반려/연장·가입 승인/거절·이벤트 개설/변경/취소·신고 처리·승계·강퇴)→수신자→딥링크 목적지 매핑 표.
- `빈_상태`: 화면별 빈 상태+역할별 CTA(모임장=생성 유도, 부원=대기 안내) 카탈로그.

## 3. 피드 도메인 (확정 · F1~F4 — 담당 해찬)

- **F1 피드 메인**: 소스 = ① 내 모임 게시글(공지·정모/번개 모집 자동 카드·일반 글) ② 팔로우한 사람 게시글. 카드에 좋아요·댓글 수, 좋아요 즉시 토글, 소스 라벨(왜 보이는지). 카드 탭→게시글 상세/이벤트 상세/모임·프로필. 빈 상태(모임 0·팔로우 0)→탐색·팔로우 유도.
- **F2 게시글 상세**: 피드·모임 게시판 **공용 화면**. 본문·미디어·좋아요·댓글(내 댓글 수정/삭제)·내 글 수정/삭제 진입·⋮ 신고 시트(사유→접수).
- **F3 글 작성/편집**: 부원 — 텍스트+사진(복수). **공지 모드**는 모임장·관리자 전용(상단 고정+부원 알림). 모집 카드는 수동 작성 대상 아님(이벤트 개설 시 자동).
- **F4 신고·모더레이션**: 부원 신고(게시글·댓글) → 관리자 신고함에서 숨김/삭제/경고, 처리 결과 신고자 알림, 처리 이력 보관. ⚠플랫폼 운영자 에스컬레이션 경계는 팀회의.

## 4. 홈 도메인 (제안본 · 원저자 예경 확정 필요)

- **홈_대시보드**: 진입(온보딩 직후·탭·복귀)→인사+"오늘 챙길 일 N건"→세로 스캔(todo→일정→내 모임→추천)→분기. 홈 재선택 시 스크롤·상태 유지+데이터만 재조회(A3·P4). 신규 사용자는 빈 상태+추천 카드(D6 콜드스타트).
- **오늘_챙길_일**: 회비 액션 큐 — 정렬(P2): 마감 있는 항목 잔여시간순→마감 없는 항목 최신순. 갱신(P4): 푸시·포그라운드 재조회·당겨서 새로고침(폴링 없음). `송금` 카드→이벤트 상세 경유→납부 흐름(모임 도메인과 동일 화면). ⚠추가정산·환급 카드는 #10(사후 정산이면 소멸).
- **다가오는_일정**: 주간 캘린더 스트립 좌우 스크롤(점=일정)+날짜 탭 시 하단 리스트 인라인 필터(별도 캘린더 화면 없음 — 확정). 이벤트 행→상세 핸드오프. **일정의 단일 정본**(마이 일정은 제거됨).
- **알림_확인**: 벨(빨간 점=미확인)→알림 센터(회원 소유)→항목 탭→원인 화면 딥링크→읽음. 만료 대상 폴백: 정산 결과→지난 이벤트→모임 홈.
- **모임_번개_탐색**: 태그 chip(내 주변·당장 가입 가능·이번 주 활동·관심사)→필터 탐색 · 추천 모임 카드(**상세 경유 강제** 후 가입) · 추천 번개 카드(⚠무소속 번개 #10) · 추천 카드는 내 모임 0일 때만(D6), 이후 태그만. ⚠탐색 화면 소유 #10.
- **글로벌_내비**: 탭바 = 홈·피드·FAB·채팅(구 DM)·마이. FAB `+` = 글쓰기/모임 만들기/정모 개설(권한자만 노출)/번개(D7). 내 모임 카드→모임 홈, `전체보기`는 소속 5개+일 때만(D5).
- **총무_운영_진입**: 권한자(모임장·관리자·번개 주최 부원)에게 todo 추가 노출 — "송금 확인 N건"·"미납 추적"·"정산 필요" → **모임 홈 경유** 관리 진입(D2 — 다중 모임 컨텍스트 혼동 방지). ⚠OCR·환급 단계는 #10.

*(제안본 크리틱 잔여 — 오너 결정 대기: 키워드 검색 부재 · 딥링크 라우팅 규정 · 번개 주최 부원 todo · 가입 심사 todo · FAB 번개의 모임 선택)*

## 5. 회원 도메인 (제안본 · 원저자 민준 확정 필요)

- **온보딩**: 소셜 로그인 3종(카카오·구글·애플) → 기존 회원은 홈 직행, 신규는 **전화번호 인증(1회)** → 전역 닉네임 → 관심사 선택(홈 추천의 데이터 소스) → **홈 빈 상태 랜딩**(추천 카드).
- **마이페이지_개요**: 정체성·기록·설정 허브 — 프로필(아바타·닉네임·모임 레벨 배지·게시글/팔로워/팔로잉 통계)·최근 정산 내역·설정 진입. 보낼/받을 돈·일정은 홈으로 이동(중복 제거).
- **모임_레벨**: 정산 이행 결과의 표시 전용 지표 — 캐릭터·상태명·점수. 상태 3종: 신규 `첫 모임·이력 쌓는 중`(낮은 점수≠불이행) / 성장 `정산 N회 완료` / 지연 `정산 확인 필요`(지연 일수+회복 안내). **공개**(상대 프로필에도 표시). 점수 산정 입력=정산 도메인 이벤트(마감·완료·지연), 편집 불가.
- **프로필_편집**: 사진·닉네임·자기소개·연락처 수정, 즉시 반영. **실명은 본인만 조회**(타인에겐 닉네임만).
- **소셜_프로필**: 내 게시글 목록(피드 소스의 내 것 필터)·팔로워/팔로잉 목록(**전체 공개**, 비공개 계정 MVP 제외)·상대 프로필(팔로우/언팔·게시글·모임 레벨 표시).
- **정산_내역**: 개인 돈 기록 — 송금·수금·환불을 기간·모임 필터로 조회(원화, 상태 배지), **영구 보관+CSV 내보내기**. 모임 아카이브의 개인 단면.
- **알림_센터**: 전 도메인 알림 단일 목록(시간순+종류 배지, 탭 분리 없음) — 개별/전체 읽음, 유사 그룹핑(좋아요 N건=1줄), **보관 30일**, 딥링크+만료 폴백. 트리거 정의는 각 도메인 매핑 표 참조.
- **개인_채팅**: 1:1 — **시작 제한 없음**(차단이 안전장치 ⚠팀회의). 목록은 상시방·이벤트방과 통합 표시(최신 메시지순).
- **설정**: 알림 토글(정산 필수 알림은 **끌 수 없음+사유 표시**, 독촉·소셜만 선택) · 계정(카카오/구글/애플 연결·전화번호·**계좌 예금주 인증** — 폴백 수취용) · 로그아웃 · 회원 탈퇴(**내가 주최한 미마감 정산만 차단** — 이관·마감 후 가능, 참여자 미납은 차단 안 함).

*(잔여 — 팀회의: 사용자 차단·신고(사람)·CS 채널 · 정산 명단 실명 여부)*
