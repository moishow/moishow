# 내 모임 일정 (주간 캘린더)

> 진입: 마이페이지 "내 모임 일정" 섹션 ([[mypage-overview]])

## User Flow
1. 마이 "내 모임 일정" — **주간 캘린더**(이번 주 7일, 오늘 강조, 일정 있는 날 마커)
2. 이전/다음 주 네비게이션으로 주 이동
3. 아래 "다가오는 일정" 리스트에서 가까운 일정 확인 (날짜 뱃지 · 모임명 · 시간 · 장소)
4. 일정 탭 → **일정 상세** (모임/차수 상세, 참석·정산 연결)
5. "일정 전체" → 전체 일정 화면(월간 등)

## 연결되는 페이지
- 마이페이지 (진입점 · 주간 뷰)
- 일정 상세 (`#schedule-detail`) — 모임/차수 상세로 연결
- 일정 전체 (`#schedule`)

## 핵심 기능 (필수)
- 내가 신청·참여한 모임 일정만 집계
- 주간 캘린더 + 다가오는 일정 리스트
- 일정 → 해당 모임/차수(정산) 연결

## 비고 (검증 필요)
- 일정의 소스: "신청한(승인 대기)" vs "확정된" 일정 시각 구분 필요 여부.
- 여러 모임 일정이 겹칠 때 마커 색/모임 구분.
- "일정 전체"를 월간 캘린더로 줄지, 리스트로 줄지.
- 마이 vs 홈 도메인의 일정 노출 중복 정리(홈에도 일정이 있다면).
