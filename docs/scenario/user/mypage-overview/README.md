# 마이페이지 진입 · 개요 (허브)

> Base 화면: [`design/mypage/mypage/mypage.html`](../../../../design/mypage/mypage/mypage.html)
> 모델 전제: **카카오페이 기반 회원 간 직접 송금 + 송금 여부 직접 체크** (앱 포인트·에스크로 없음).

## User Flow
1. 하단 탭바 "마이" 탭 진입
2. 한 화면에서 아래를 순서대로 확인:
   - **프로필**: 아바타 · 이름 · 자기소개 · 모임 레벨 배지 · [편집] · 통계(게시글/팔로워/팔로잉)
   - **보낼 돈**: 미송금 정산 강조 행 (붉은 알림)
   - **받을 돈**: 내가 주최한 걷기 현황 + 환급 예정
   - **내 모임 일정**: 주간 캘린더 + 다가오는 일정
   - **최근 정산 내역**: 송금/환급 상태 배지
3. 각 영역을 탭해 하위 흐름으로 이동

## 마이페이지에서 **빠진** 것 (경계 정리)
- **내 모임 목록** → 홈 도메인에서 노출 (마이에 중복 배치 안 함)
- **카카오페이 연결 메뉴** → 본인 카카오페이 앱이 직접 열리는 방식이라 불필요
- **포인트 잔액 / 충전 / 현금화** → 포인트 모델 폐기로 제거

## 비고 (검증 필요)
- 마이페이지는 "소셜 프로필 + 정산 허브"의 결합. 상단은 소셜(프로필·팔로우), 중단은 정산(보낼/받을 돈)으로 성격이 갈림 — 이 결합이 맞는지, 정산만 별도 탭으로 뺄지 검증.
- 모임 레벨은 **표시만** 담당. 상세 상태·회복 안내는 [[trust-tier]]에서 확인하며, 점수 산정 규칙(정산 지연 반영)은 정산/총무 도메인 소관.
