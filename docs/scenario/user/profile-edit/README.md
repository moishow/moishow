# 프로필 조회 · 편집

> 진입: 마이페이지 프로필 [편집] 버튼 ([[mypage-overview]])

## User Flow
1. 마이 프로필 영역 [편집] → **프로필 편집**
2. 수정: 프로필 사진 · 닉네임 · **자기소개(한 줄)** · 연락처
3. 저장 → 마이페이지 복귀, 즉시 반영

## 연결되는 페이지
- 마이페이지 (진입점)
- 프로필 편집 (`#profile-edit`)

## 핵심 기능 (필수)
- 닉네임 · 프로필 사진 · 자기소개 편집
- 저장 후 프로필 헤더 즉시 갱신

## 비고 (검증 필요)
- **KYC 실명/CI**는 편집 불가 — 조회만 노출할지, 숨길지. 재인증은 가입·인증 도메인(다른 팀원) 소관.
- 프로필 사진 부적절 이미지 신고·모더레이션 대상 여부(쇼 도메인과 연계).
- 신용등급은 편집 불가(표시 전용).
