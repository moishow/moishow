# 보낼 돈 — 카카오페이 송금 + 송금 여부 직접 체크

> 진입: 마이페이지 "보낼 돈" 강조 행 또는 "정산 전체" ([[mypage-overview]])
> 방향: 부원 → 총무(회비·추가정산), 부원 → 부원(개인 벙 정산), **주최자 → 부원(참가 취소 환불)**

## User Flow (공통)
1. 마이 "보낼 돈" 붉은 행 탭 → **송금 상세**
2. 정산 항목 확인: 금액 · 받는 사람(카카오페이 계정) · 사유
3. **[카카오페이로 보내기]** → 본인 카카오페이 앱 열림 → 송금
   - 카카오페이가 안 열리면 **계좌 이체 폴백**(받는 사람 계좌 복사)
4. 앱 복귀 후 **"송금 완료했어요" 직접 체크**
5. 상태: `미송금` → (체크) `확인 대기` → (받는 사람이 입금 확인) `입금 확인`
6. 완료 시 마이 "보낼 돈"에서 사라지고 "최근 정산 내역"으로 이동

## 보낼 돈의 3가지 사유 — **송금 상세(pay-detail) 화면을 재사용**
| 사유 | 보내는 사람 → 받는 사람 | 배지 |
|---|---|---|
| 회비 송금 | 부원 → 총무 | `미송금` |
| 추가 정산 | 부원 → 총무 | `미송금` |
| **참가 취소 환불** | **주최자 → 취소한 부원** | `환불 요청` |

> 사유·받는 사람·모임만 달라지고 **송금 + 송금완료 체크 흐름은 동일** → 환불 전용 페이지를 따로 만들지 않는다.
> 받는 쪽(취소한 부원)에서는 `환불 대기`로 보인다 ([[receive-money]]).

## 연결되는 페이지
- 마이페이지 (진입점)
- 송금 상세 (`#pay-detail`) — 카카오페이 보내기 · 송금완료 체크
- 정산 전체 (`#settlements`) — 보낼 돈 목록
- 정산 내역 ([[settlement-history]]) — 완료 후 기록

## 핵심 기능 (필수)
- 미송금 항목 강조 노출 (마이 메인에는 금액·상태만, 체크는 상세에서)
- 카카오페이 앱 딥링크 송금
- 송금 완료 자가 체크 → 상태 전이

## 비고 (검증 필요)
- **자가 체크의 신뢰성**: 부원이 안 보내고 체크할 수 있음 → 받는 사람(총무/주최자) 입금 확인이 최종 확정. 이중 확인 흐름은 [[receive-money]]와 맞물림.
- **미납 지속 시**: 강제출금 불가 정책 → 프로필 표시·지속 알림만 ([[settings]] 알림). 신용등급 반영 여부.
- 카카오페이 딥링크 실패/미설치 시 폴백(계좌번호 복사 등) 필요 여부.
