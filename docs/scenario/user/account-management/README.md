# 계정 관리 · 로그아웃 · 회원 탈퇴

> 진입: 설정의 계정 설정 ([설정](../settings/README.md))

## User Flow A — 계정 정보 확인

1. 계정 설정에서 카카오 로그인, 전화번호 인증, 등록 계좌 상태를 확인한다.
2. 계좌 등록·인증을 선택하면 계좌 관리 흐름으로 이동한다.

## User Flow B — 로그아웃

1. 로그아웃을 선택한다.
2. 확인 후 세션을 종료하고 온보딩(W-01)으로 돌아간다.

## User Flow C — 회원 탈퇴

1. 회원 탈퇴를 선택한다.
2. 미송금, 받을 돈, 진행 중 일정이 있으면 먼저 정리해야 함을 안내한다.
3. 조건 충족 시 탈퇴 사유와 최종 확인을 받은 뒤 탈퇴를 요청한다.

## 연결되는 페이지

- 계정 설정 (`design/mypage/account-settings/account-settings.html`)
- 계좌 등록·인증 (`design/mypage/bank-account/bank-account.html`)
- 온보딩 (W-01)

## 핵심 기능

- 연결·인증 상태 표시
- 로그아웃 확인 및 세션 종료
- 탈퇴 가능 여부 사전 검사

## 비고 (검증 필요)

- 탈퇴 뒤 KYC/CI와 거래 기록의 보관·파기 정책은 인증·정산 도메인과 합의한다.
