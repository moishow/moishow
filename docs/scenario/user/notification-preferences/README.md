# 알림 설정

> 진입: 설정의 알림 설정 ([설정](../settings/README.md))

## User Flow

1. 설정에서 알림 설정으로 진입한다.
2. 정산, 송금 요청, 미납·독촉, 소셜 알림의 현재 수신 상태를 확인한다.
3. 사용자가 선택 가능한 카테고리를 켜거나 끈다.
4. 변경을 즉시 저장하고 설정으로 돌아간다.

## 연결되는 페이지

- 설정 (`design/mypage/settings/settings.html`)
- 알림 설정 (`design/mypage/notification-settings/notification-settings.html`)

## 핵심 기능

- 카테고리별 수신 동의 상태 조회·변경
- 변경 실패 시 이전 상태 복원 및 오류 안내

## 비고 (검증 필요)

- 송금 요청과 입금 확인을 필수 알림으로 둘 경우 토글 비활성 사유를 화면에 표시한다.
- OS 알림 권한이 꺼진 경우 시스템 설정으로 연결하는 흐름이 필요하다.
