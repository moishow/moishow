# 소셜 프로필 — 팔로워 · 팔로잉 · 내 게시글

> 진입: 마이페이지 프로필 통계(게시글/팔로워/팔로잉) ([[mypage-overview]])

## User Flow
1. 마이 프로필 통계 탭:
   - **게시글 N** → 내가 쓴 쇼 게시글 목록
   - **팔로워 N** → 나를 팔로우한 사용자 목록
   - **팔로잉 N** → 내가 팔로우한 사용자 목록
2. 목록에서 사용자 탭 → 상대 프로필 (팔로우/언팔로우)
3. 게시글 탭 → 쇼 상세(W-33)

## 연결되는 페이지
- 마이페이지 (진입점 · 개수만 표시)
- 내 게시글 목록 (`#my-posts`) → 쇼 상세 (기존 W-33)
- 팔로워 목록 (`#followers`) / 팔로잉 목록 (`#following`)
- 상대 프로필

## 핵심 기능 (필수)
- 게시글/팔로워/팔로잉 개수 표시 + 목록 진입
- 팔로우/언팔로우

## 비고 (검증 필요)
- **팔로우 ↔ DM 관계**: 팔로우가 DM(W-29/30) 가능 조건인지, 상호 팔로우 필요인지.
- 내 게시글은 쇼 도메인(W-27/28/33)과 소스 공유 — 마이는 "내 것만 필터" 조회.
- 차단/신고한 사용자 관리가 소셜 프로필에 필요한지(모더레이션 도메인 연계).
- 팔로워/팔로잉 공개 범위(비공개 계정 여부).
