# 정산 내역 조회

> 진입: 마이페이지 "최근 정산 내역 → 전체보기" ([[mypage-overview]])

## User Flow
1. 마이 "최근 정산 내역"(3건 미리보기) → "전체보기" → **정산 내역 전체**
2. 송금·환급·수금 내역을 기간·모임별로 조회
3. 각 항목 상태 배지 확인: `입금 확인`(완료) / `확인 대기`(보냈으나 미확인) / `미송금`
4. 개별 항목 탭 → 상세(일시 · 금액 · 상대 · 모임/차수 · 카카오페이 여부)

## 연결되는 페이지
- 마이페이지 (진입점)
- 정산 내역 전체 (`#wallet-ledger`)
- 항목 상세

## 핵심 기능 (필수)
- 원화 기준 송금/환급 내역 목록 (포인트 아님)
- 상태 배지로 송금 여부 추적
- 기간·모임 필터

## 비고 (검증 필요)
- **기록의 출처**: 카카오페이 P2P라 앱은 "송금했다고 체크된" 기록만 보유(실 이체는 카카오페이). 자동 대사 불가 → 분쟁 시 근거로 어디까지 인정할지.
- 상대방 입금 확인 전/후 기록을 어떻게 구분해 보여줄지 (확인 대기 vs 확정).
- 내역 보관 기간·CSV 내보내기 필요 여부.
