# 계좌 등록 · 인증

> 진입: 계정 설정의 계좌 등록·인증 ([계정 관리](../account-management/README.md))

## User Flow

1. 등록된 계좌와 인증 상태를 확인한다.
2. 은행과 계좌번호를 입력해 인증을 요청한다.
3. 인증 코드 또는 예금주 확인을 수행한다.
4. 성공하면 인증됨 상태로 계정 설정에 반영한다.
5. 실패하면 사유를 안내하고 재입력 또는 재인증을 제공한다.

## 연결되는 페이지

- 계정 설정 (`design/mypage/account-settings/account-settings.html`)
- 계좌 등록·인증 (`design/mypage/bank-account/bank-account.html`)

## 핵심 기능

- 등록 계좌·예금주·인증 상태 표시
- 입력값 형식 검증과 인증 요청
- 실패·재시도·계좌 변경
