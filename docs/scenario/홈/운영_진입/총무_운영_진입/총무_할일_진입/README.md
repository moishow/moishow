# 총무_할일_진입 — 운영 todo 처리 진입

> 상위: [총무_운영_진입](../README.md) · **⚠일부 충돌(#10)**

## User Flow (원저자안 보존)
1. 홈 todo의 권한자 전용 카드: "송금 확인 N건" · "미송금 추적" · "정모 종료·정산 필요"
2. 탭 → 납부 현황판/정산 생성으로 핸드오프(모임 도메인)

## 분기 / 엣지
- 송금 확인·미송금 추적은 모임 스펙 납부_추적과 **수렴** ✅
- "OCR→자동 정산→환급/추가정산"(원저자 W-19·21~23 흐름)은 **⚠충돌(#10)** — 사후 정산이면 "정산 만들기(실지출+영수증)"로 대체
- 다중 모임 권한자: todo에 모임 컨텍스트 표기 필요(원저자 노트)
- 역할 표기 "총무" → 모임 스펙 v10 "모임장·관리자" 전파 필요(경미)
