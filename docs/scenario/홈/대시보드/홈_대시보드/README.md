# 홈_대시보드 — 진입·개관 인덱스

> 상위: [홈 도메인](../../README.md) · 원 시나리오: view-dashboard(예경)

## 개요
앱 진입/재진입 시 내 상태를 한 화면으로 파악하고 다음 행동으로 분기. 홈은 "조회" 시나리오라 자체 종점이 없고 모든 실질 행동을 다른 시나리오에 위임한다(원저자 노트).

| [빈_상태_온보딩](빈_상태_온보딩/README.md) | 신규 사용자 | 데이터 0 → 탐색 유도 |
