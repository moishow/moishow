## User Flow (원저자안 보존)
1. 추천 번개 카드 탭 → 번개 상세
2. 참여 신청 → (회비 있으면) 회비 송금 흐름
3. **무소속 번개**는 모임 홈이 없어 번개 상세가 종점 컨텍스트

## 분기 / 엣지
