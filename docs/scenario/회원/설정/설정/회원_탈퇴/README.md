# 회원_탈퇴 — 사전 검사·탈퇴

> 상위: [설정](../README.md)

## User Flow
1. 탈퇴 선택 → **사전 검사**: 미송금·받을 돈·진행 중 일정 있으면 정리 안내(차단)
2. 조건 충족 → 사유 입력 → 최종 확인 → 탈퇴 요청

## 분기 / 엣지
- (논의·원저자) 미정산 잔존 시 탈퇴 허용 여부는 정산 도메인과 조율 → 모임 스펙 유사 결정 참고: 모임 탈퇴=미납 고지 후 허용·명단 유지. **회원 탈퇴는 계정 소멸이라 더 엄격 필요**(주최 중 정산은 차단 권장 — 제안)
- 데이터 보관·파기(CI 등)는 인증 도메인 소관(원저자) · 탈퇴자의 모임 내 기록은 "실명 유지+탈퇴 표시"(모임 스펙 확정)와 정합 확인
