# 계좌_등록_인증 — 실계좌 인증

> 상위: [설정](../README.md)

## User Flow (원저자안 보존)
1. 등록 계좌·인증 상태 확인 → 은행·계좌번호 입력 → 인증 요청
2. 인증 코드/예금주 확인 → 성공 시 `인증됨` 반영 · 실패 시 사유 안내·재시도

## 분기 / 엣지
- **(논의) 필요성 재확인 권장**: 송금이 전부 카카오페이 딥링크(외부)라면 앱 내 계좌 등록의 용도가 무엇인지 — 폴백(계좌 복사)용 수취 계좌 표시라면 "인증"까지 필요한가. 외부 인증 API 비용·부담 대비 검토(포인트용 KYC 폐기 취지와 연결)
- 외부 API·재시도 횟수·변경 이력 보관 책임(원저자 검증 항목)
