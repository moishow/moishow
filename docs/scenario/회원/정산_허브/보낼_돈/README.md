# 보낼_돈 — 송금 요약·실행 인덱스

> 상위: [회원 도메인](../../README.md) · 원 시나리오: send-money(민준)

## 개요
마이의 미송금 정산 강조 목록 → 송금 상세(재사용 화면)에서 카카오페이 송금+셀프 체크. 원저자 확정: 사유(회비/추가정산⚠/환불)가 달라도 **송금 상세 화면 하나를 재사용**(환불 전용 페이지 없음).
