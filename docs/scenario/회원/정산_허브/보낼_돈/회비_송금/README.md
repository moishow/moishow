# 회비_송금 — 내 몫 보내기

> 상위: [보낼_돈](../README.md)

## User Flow
1. 마이 "보낼 돈" 붉은 행 탭 → 송금 상세(금액·받는 사람·사유)
2. [카카오페이로 보내기] → 외부 송금 (실패 시 폴백: 계좌번호 복사 — 원저자안, 모임 스펙과 정합 ✅)
3. 복귀 → "송금 완료했어요" 셀프 체크 → `확인 대기` → 받는 쪽 확인 → `입금 확인`
4. 완료 시 보낼 돈에서 제거 → 정산 내역으로 이동

## 분기 / 엣지
- 모임 스펙 회비_납부(납부_실행)와 **동일 흐름(수렴 ✅)** — 진입점만 다름(마이 vs 홈 todo vs 이벤트). 화면 구현 시 단일 화면
- 셀프 체크 신뢰성: 받는 쪽 확인이 최종(원저자 노트 — 모임 스펙 2단계 확인과 동일)
