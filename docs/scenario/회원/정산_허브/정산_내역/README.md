# 정산_내역 — 내 돈 기록 조회 (leaf)

> 상위: [회원 도메인](../../README.md) · 원 시나리오: settlement-history(민준)

## 행위자
본인

## 트리거 / 진입
마이 "최근 정산 내역(3건)" → 전체보기

## User Flow
1. 송금·수금·환불 내역을 기간·모임별 조회 (원화 기준 — 포인트 아님, 원저자 확정)
2. 상태 배지: `입금 확인`/`확인 대기`/`미송금`
3. 항목 탭 → 상세(일시·금액·상대·모임/정모)

## 분기 / 엣지
- 원저자 핵심 노트: "기록의 출처는 '체크된' 기록뿐(실이체는 카카오페이) — 분쟁 근거 한계" → 모임 스펙의 타임스탬프 근거·아카이브와 **상호 보완**(개인 뷰 vs 모임 뷰)
- (논의·원저자) 보관 기간·CSV 내보내기 · 용어 "차수"→"정모" 치환 필요
- 모임 스펙 정산 아카이브와 데이터 소스 공유 확인(⚠경계 — 같은 기록의 개인 단면)
