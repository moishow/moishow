# 환급_받기 — 잉여 환급 수령 ⚠충돌

> 상위: [받을_돈](../README.md) · **⚠충돌(#10)**

## 개요
원저자 모델: 총무 정산 완료 → 잉여 균등 환급 → `지급 대기`→`입금 확인`. **모임 스펙(사후 정산)에는 잉여 환급이 없어 소멸** — 정산 정정 차액 환불 수령으로 축소 존치 가능.

## User Flow (원저자안 보존)
1. 환급 예정 행 노출(`지급 대기`) → 총무 카카오페이 송금 → 입금 확인 → 정산 내역 `+금액`
