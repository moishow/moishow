# 받을_돈 — 수금·수령 인덱스

> 상위: [회원 도메인](../../README.md) · 원 시나리오: receive-money(민준)

## 개요
내가 받을 돈 세 종류: ① 주최자로서 걷기(수금 확인) ② 환급 받기⚠ ③ 환불 받기. 원저자 노트: "걷기 주체는 총무뿐 아니라 개인 벙 연 부원도" — 모임 스펙 주최자 모델과 **수렴** ✅.
