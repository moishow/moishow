# 마이페이지_개요 — 허브 열람·분기 (leaf)

> 상위: [회원 도메인](../../README.md) · 원 시나리오: mypage-overview(민준)

## 개요
마이 탭 진입 → 프로필·보낼 돈·받을 돈·일정·정산 내역을 한 화면에서 확인하고 하위 흐름으로 분기.

## 행위자
전원

## 트리거 / 진입
탭바 `마이`

## User Flow
1. 프로필(아바타·이름·모임 레벨 배지·통계) 확인
2. 보낼 돈(미송금 강조) → 보낼_돈 · 받을 돈 → 받을_돈
3. 내 모임 일정 → 내_일정 · 최근 정산 내역 → 정산_내역
4. 톱니바퀴 → 설정

## 분기 / 엣지
- 원저자 검증 항목: 소셜+정산 허브 결합의 타당성(정산만 별도 탭으로 뺄지)
- 보낼/받을 돈 항목 구성은 ⚠충돌(#10) 종속(환급·추가정산 존폐)
