# 마이페이지_개요 — 허브 열람·분기 (leaf)

> 상위: [회원 도메인](../../README.md) · 원 시나리오: mypage-overview(민준)

## 개요
마이 탭 진입 → 프로필·보낼 돈·받을 돈·일정·정산 내역을 한 화면에서 확인하고 하위 흐름으로 분기.

## 행위자
전원

## 트리거 / 진입
탭바 `마이`

## User Flow
1. 프로필(아바타·이름·모임 레벨 배지·통계) 확인
2. 최근 정산 내역(개인 기록) → 정산_내역 · 내 모임 일정 → 내_일정
3. 톱니바퀴 → 설정
※ 보낼/받을 돈 요약은 **홈으로 이동(리뷰 결정 1)** — 마이에서 제거

## 분기 / 엣지
- **결합 질문 해소(리뷰 결정 1)**: 돈 현황=홈, 마이=정체성·기록·설정 — 원저자 검증 항목("정산만 뺄지")에 대한 답. 예경 P3 번복 제안이라 팀 조율 필요(#10)
