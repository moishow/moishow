# 레벨_확인 — 상태·점수 열람

> 상위: [모임_레벨](../README.md)

## User Flow
1. 마이 프로필의 레벨 배지 탭 → 상세
2. 캐릭터·상태명·점수·진행 막대·다음 단계 확인
3. 이력 없는 신규는 `첫 모임 · 이력 쌓는 중`(낮은 점수 ≠ 불이행 — 원저자 UI 계약)

## 분기 / 엣지
- 점수는 편집 불가(원저자 확정) · **공개 범위 = 공개(리뷰 결정 2)** — 상대 프로필에도 표시, 신규는 `이력 없음` 구분
- (논의·원저자) 분쟁·장애·예외의 감점·회복 기준 미정 — 확정 전 `확인 중` 상태 분리
- 모임 스펙과 접점: 미납·지연 이벤트가 입력 — 정산 도메인과 이벤트 스키마 협의(⚠경계)
